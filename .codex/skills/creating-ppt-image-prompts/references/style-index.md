## Selection Rules

- Match the communication job and audience before decoration.
- Brand profiles are visual direction and never imply official authorization.
- If no profile is a strong match, use `bright-blue-purple-academic` or define a user-specific profile in deck metadata.
- Read exactly one profile unless the user explicitly requests a hybrid; document the source rules of any hybrid.
